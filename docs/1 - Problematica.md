# Resumen del Proyecto

## Sistema Web de Gestión Académica y Comunicación Escolar

**Institución:** Unidad Educativa Oswaldo Guayasamín (Galápagos)

### Problemática Actual
La institución maneja sus procesos de forma manual usando hojas de cálculo, WhatsApp y documentos físicos, lo que genera:
- Retrasos en la entrega de calificaciones
- Pérdida de información
- Duplicidad de registros
- Falta de trazabilidad
- Comunicación informal e ineficiente entre docentes, padres y estudiantes

### Solución Propuesta
Desarrollar una plataforma web centralizada que:
- Automatice el registro de calificaciones, tareas y asistencias
- Facilite la comunicación entre docentes, padres y estudiantes
- Proporcione acceso en tiempo real a la información académica
- Genere reportes académicos
- Funcione eficientemente incluso con conectividad limitada (contexto insular)

### Usuarios del Sistema
1. **Docentes**: Registran calificaciones, tareas, asistencias
2. **Padres**: Consultan información académica de sus hijos, reciben notificaciones
3. **Estudiantes**: Consultan sus calificaciones, tareas pendientes
4. **Administradores**: Gestionan usuarios, cursos, configuraciones del sistema

### Stack Tecnológico Propuesto
- **Backend**: Laravel + Node.js
- **Frontend**: Tailwind CSS
- **Base de Datos**: MySQL
- **Metodología**: Scrum
